# SaaS 플랫폼 기획 & 마케팅 전략

이 문서는 MassageShop 프로젝트의 다중 샵 SaaS 모델 설계, 고객 유입 식별, 앱/QR 정책, 비콘 마케팅 등 주요 기획 내용을 정리한다. 지속적으로 업데이트하여 프로젝트의 설계 방향과 마케팅 아이디어를 기록한다.

---

## 1. 멀티 샵 & 카테고리 SaaS 설계

### 요구 사항
- 동일 업종(마사지, 헤어, 네일 등)에서 여러 샵을 독립적으로 운영
- 샵마다 브랜드, 테마, 관리자 권한, 결제/정산 구조 분리

### 데이터 모델
- `shops` 테이블
  - 필드: `id`, `slug`, `category`, `name`, `ownerId`, `currency`, `theme`, `status`, 등
  - `locations` 또는 `branches` 관계로 분기 가능
- 예약, 결제, 리포트는 모두 `shopId` 기반 다중 테넌시

### URL & 라우팅
- 기본 경로 `/{category}/{shopSlug}` (예: `/massage/lotus`, `/hair/kbarber`)
- `ShopContext`에서 카테고리/슬러그를 감지해 테마·문구 적용

### 관리자/정산
- 샵 오너의 전용 대시보드
- 플랫폼 운영자는 전체 샵 통합 뷰와 정산 기능 보유
- 샵별 수수료율, 결제 수단 설정 가능

---

## 2. 유입 혼선 해결 & 정산 식별

### 목표
- 고객과 샵 오너의 진입 경로 혼선 제거
- 플랫폼의 매출 정산용 식별 구조 마련

### 설계
1. **고유 샵 코드 발급**
   - 샵별 `shop_code` 생성 (예: `MSG001`, `HAIR-KIM01`)
   - QR/바코드에 코드와 카테고리를 암호화 포함
2. **입장/예약 시 파라미터 전달**
   - QR 스캔 시 `?shop=MSG001` 또는 헤더에 코드 삽입
   - SPA 초기화 때 해당 값으로 `ShopContext` 설정
   - 쿠키/세션에 `source=qr|web|app` 저장
3. **백오피스 & 로그**
   - API 로그에 `shop_code`, `source` 기록
   - 샵 오너 리포트: 채널별 유입/결제 통계
   - 정산 모듈은 `shop_code` 기준 매출·수수료 계산

## 3. QR/앱 접근 정책

### QR 코드
- 샵 관리자 대시보드에서 고유 QR 생성
- 내용: `https://{domain}/{category}/{shopSlug}?ref=qr&code={shop_code}`
- 인쇄물, 웹, SNS 등에 활용

### 앱 사용 흐름
- Play/앱스토어에 단일 앱 배포
- 설치 후 딥링크 처리:
  - QR→앱 경로로 `ShopContext` 자동 활성화
  - 일반 실행 시 샵 선택 화면 표시 (최근 방문 샵 제안)
- 앱 내 즐겨찾기 샵 등록 기능

### 사용자 안내
- QR 옆에 설명 문구 기재
- 앱 온보딩에서 비콘/QR 알림 UX 안내

## 4. 비콘 기반 마케팅 팝업

### 개념
- 샵 근처를 지나는 고객에게 자동 팝업/알림 제공
- BLE 비콘 설치 + 모바일 앱 연동

### 아키텍처
1. **비콘**
   - 출입구 또는 거리마다 설치
   - UUID/major/minor와 `shop_code` 매핑
2. **모바일 앱**
   - 백그라운드 BLE 스캔 (Android Foreground Service, iOS Region Monitoring)
   - 비콘 감지 시 로컬 푸시 또는 앱 팝업 발송
3. **서버/관리 콘솔**
   - 샵별 프로모션 메시지 등록
   - 비콘 ID에 메시지 매핑
   - 트리거 통계 수집

### 동작
1. 사용자가 앱 설치·권한 허용
2. 비콘 신호 범위에 진입 시 로컬 알림 혹은 팝업
3. 메시지 클릭 시 예약 페이지로 이동하거나 쿠폰 저장
4. 이벤트 로그 서버 전송(비콘, 시간, 사용자)

### 구현 고려
- iOS: 항상 허용 위치 권한 필요, 백그라운드 스캔 제약
- Android: 배터리 최적화, Foreground Service
- 개인정보: 위치/블루투스 정보 동의・퀀텐츠 정책 문서화
- 하드웨어 관리: 비콘 배터리·위치·등록 UI

### 마케팅 시나리오
- “지금 지나가시나요? 10% 쿠폰 드려요!”
- “앱으로 예약하면 추가 혜택”
- “3일 연속 방문자 전용 프로모션”
