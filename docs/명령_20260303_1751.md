### 📝 수행 적요 (Summary)
- 사용자의 "아직 언어선택이 반영이 안되었어" 피드백 접수.
- 즉시 프론트엔드(`LandingPage.tsx`)에 다국어(한국어, English, 日本語, 中文) 지원을 위한 **글로벌 언어 드롭다운(Language Selector)** UI 컴포넌트 추가 완료.
- `lucide-react`의 Globe 아이콘 적용 및 `framer-motion`을 활용한 부드러운 애니메이션 효과를 부여하여 UI의 완성도를 한층 높였습니다. 
- 터미널을 통한 프론트엔드 HMR(Hot Module Replacement) 자동 반영으로 로컬 서버(`localhost:5173`)에서 곧바로 확인 가능합니다.
